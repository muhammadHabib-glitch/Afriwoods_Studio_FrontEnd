# Afriwood Studios — Static Assets

Brand images served from `/assets/` in the Next.js app.

## Structure

```
public/assets/
├── heroes/        Character portraits
├── comics/        Comic cover art
├── products/      Shop product images
├── news/          News thumbnails
├── backgrounds/   Section backgrounds
├── ui/            Logos and UI elements
└── pages/         Design reference screenshots (optional)
```

## Notes

- Image paths in code use `/assets/...` (e.g. `/assets/heroes/shutter-bird.png`).
- Raw Figma exports (`figma-raw/`) are excluded from git — keep them local or in separate storage.
- Replace or add files under the folders above; no code changes needed if filenames stay the same.
